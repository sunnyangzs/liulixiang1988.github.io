    1. @media

    sass中的@media跟CSS区别：

    sass中的media query可以内嵌在css规则中，在生成CSS的时候， media query 才会被提到样式的最高级

    好处：避免了重复书写选择器或者打乱样式表的流程。

    @mixin col($width){
    @media(min-width: 768px){
        width: $width;
        float: left;
    }
}

    2. @at-root

    指定嵌套的内容生成CSS时，在最底层

            font:{
                family: $main-sec-ff;
                size: 16px;
            }
        }

        .main-sec-detail{
            font-size: 12px;
        }
    }
}
