6. 页码制作

a标签是内联元素，为了使内联元素具有块级元素的一些属性同时使所有的a标签显示在同一行上，我们需要把a标签设置为行内块元素inline-block。

<div class="page">
    <a href="#">首页</a>
    <a href="#">&lt;</a>
    <a href="#">1</a>
    <a href="#">2</a>
    <a href="#">3</a>
    <a href="#">...</a>
    <a href="#">&gt;</a>
    <a href="#">末页</a>
</div><!-- page结束 -->

//页码
.page{
    height: 40px;
    margin-top: 10px;
    text-align: center;

    a{
        border: 1px solid #E8E8E8;
        text-decoration: none;
        margin: 5px;
        padding: 5px 10px;
        display: inline-block; //行内块级元素

        &:link, &:visited{
            color: #000;
        }

        &:hover, &:active{
            color: #FFF;
            background-color: #cc1b1b;
        }
    }
}
